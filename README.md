# Scanned-PDF-parser-with-OCR-All-in-one

A unified, OCR solution that combines the best open-source OCR engines into a single, easy-to-use pipeline. Parse scanned PDFs, images, and documents with multiple OCR backends for maximum accuracy and flexibility.

- Features
6 OCR Engines in One: Docling, docTR, Surya, EasyOCR, Tesseract, PaddleOCR
Intelligent Engine Selection: Auto-select best engine based on document type
Multi-Format Support: PDF, DOCX, XLSX, HTML, images (PNG, JPG, TIFF)
Advanced Capabilities:

Table extraction and structure preservation
Mathematical formula recognition (LaTeX)
Layout analysis and document understanding
Key information extraction (KIE)
100+ language support
Handwritten text recognition
